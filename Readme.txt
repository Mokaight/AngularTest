Projet de Test pour la société Afidium.

Ce Projet comprend les technologies suivantes : 
- Angular et son CLI
- NPM
- SCSS / Typescript / Bootstrap

Pour lancer le projet en local : 

Se situer dans la racine du dossier et effectuer un : 

ng serve --open 

Le site sera dès lors accessible sur localhost:4200/

Ne pas oublier : 
 - D'avoir NodeJS ( https://nodejs.org/en/download/ )
 - D'avoir npm installé 
 - D'avoir Bootstrap d'installé  (  npm install bootstrap@3.3.7 --save ) 
 - D'avoir le CLI d'Anglu installé : ( npm install -g @angular/cli  pour une installation en global)
